# Termux theme changer

This tool helps to change termux theme from the command line.

## Awailable themes

- Base16 default dark
- Base16 default light
- Base16 greenscreen dark
- Dracula
- Gnometerm new
- Gruvbox dark
- Gruvbox light
- Material
- Solarized dark
- Solarized light
- Tokyonight dark
- Tokyonight day
- Tomorrow night
- Ubuntu
- Zenburn

## Installation

Clone the repository

```
git clone https://github.com/magicisyou/termux-theme-changer
```

Change directory

```
cd termux-theme-changer
```

Make executable

```
chmod u+x termux-theme.sh
```

Run

```
./termux-theme.sh
```
